# Fuentes verificadas v3

## Georef / Datos Argentina

Base de descarga completa con recursos en CSV, JSON, GeoJSON y NDJSON. Recursos configurados:

- Provincias.
- Departamentos.
- Municipios.
- Gobiernos locales.
- Localidades.
- Localidades censales.
- Asentamientos.

Recursos opcionales probados por el downloader:

- Aglomerados.
- Fracciones censales.
- Radios censales.

## INDEC Censo 2022

Fuentes principales:

- Resumen país/provincias.
- Gobiernos locales.
- Cuadros por jurisdicción a nivel departamento/partido/comuna.
- REDATAM Censo 2022 para exportaciones por localidad y otras geografías.
- Codgeo 2022 para códigos geográficos oficiales.

## REFEGLO

Registro Federal de Gobiernos Locales. Se usa para enriquecer gobiernos locales, categoría, coordenadas, localidades administradas y población 2010.

## Barrios

- CABA: fuente oficial parcial.
- RENABAP: fuente oficial parcial para barrios populares.
- Municipios/provincias: fuente oficial local si existe.
- OSM/Geofabrik: complemento no oficial.

## Código Postal Argentino

Se usa solo como señal auxiliar para matching de fuentes externas. No reemplaza códigos INDEC/Georef.
